    Big Updates to My Blog

    3 Jul 2024
    Meta

    I’ve made some big changes to my blog! This post is just for me to document what I’ve done and why, as well as to test some of the new features I’ve added.

    First off: new fonts! I am using Valkyrie for the body text and Concourse for the headings. I’ve increased the font size on larger displays for added legibility. I also made the background a little darker for better contrast.

    Next up: margin notes. I stole the margin notes code from Tufte CSS. It’s still a little rough, but it’s getting there. I wanted to try adding margin notes to this blog for some time. Adding margin notes necessitated a whole host of changes; the biggest one is that the table of contents has moved from the right side of the page to the left column. This is to free up some space.

    I’m liking how this is shaping up. Am I aware that this is less-than-useful customization? Yes. Does it make me happy? Also yes.
